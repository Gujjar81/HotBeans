Available Courses

Here are few online available courses that are free.

Some of these start from very basic level just to get you started with web development.

Khan academy

Is a non-profit organisation and
it is creating online tools to help educate students.
It produces lessons in form of videos.

Freecodecamp

Free code camp is also a non-profit organisation that is a
interactive online platform including online forum and publications. It’s main
goal is to make learning web development accessible for everyone.

W3Schools

Is free educational website for learning coding.
It has an online text editor called “Tryit Editor”.
